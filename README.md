# Glassdoor Web Security Assessment (Educational Portfolio)

## Overview

This repository contains the results of an automated web application security assessment performed against publicly accessible resources of Glassdoor using Burp Suite Professional.

The purpose of this assessment was to gain hands-on experience with automated vulnerability scanning, analyzing security findings, and documenting potential web application security issues.

> **Disclaimer**
>
> This repository is intended for educational and portfolio purposes only.
> The findings presented here are generated by Burp Suite Professional's automated scanner and have **not been manually verified**. Automated scanners may produce false positives. No exploitation or unauthorized testing was performed.
